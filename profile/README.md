# Rewire: Discover, Stream, and Develop

rewire is a startup building developer tools for robotics. No installs,
no setup, no config files — just tools that work out of the box.

Our first product is a zero-install bridge that streams live ROS 2
topics to the Rerun viewer for real-time visualization. No ROS 2
installation required — it speaks DDS and Zenoh natively. But we're not
stopping there. We're building the toolchain robotics engineers wish
they had: visualization, introspection, deployment, and more.

We think robotics has a tooling problem. The frameworks are powerful
but the developer experience around them hasn't kept up. We're here to
fix that — one tool at a time.
